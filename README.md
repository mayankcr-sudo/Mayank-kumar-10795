Project Title

Menu Driven Program for Number Operations and Marks Grading System

Objective

This project is created to understand how functions, loops, and switch-case work together in C programming.

Tools Used

Language: C

Editor: VS Code

Compiler: GCC

Features

Prints numbers from 1 to N

Prints even numbers

Prints odd numbers

Calculates total marks

Calculates percentage

Assigns grades

Why this project was made

This project was created to improve logical thinking and to understand how to separate program logic using functions.

Utility

Helpful for beginners learning C programming and menu-driven programs.

Future Scope

Add student name and roll number

Store data using files

Create a full result management system

What I learned

How to create and call functions

How switch-case controls program flow

How loops work inside functions

How to upload projects on GitHub
